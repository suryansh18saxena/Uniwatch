💡 How Alerts Work

  1. Configure rules in the Admin Panel → Alert Rules (set CPU > 90%, Disk > 85%, etc.)
  2. Click "Check Now" to scan all active servers against your rules
  3. If a threshold is exceeded, an alert appears with a 🔧 Fix Now button
  4. Click Fix → enter SSH key → we SSH into your server and execute the fix
  5. Your SSH key is deleted immediately after use
