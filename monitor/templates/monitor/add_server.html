What happens next?
01
SSH Connection
Univatch connects to your server over port 22 using Paramiko with your provided private key.
02
Agent Deployment
Node Exporter v1.7.0 is installed and started as a systemd service. cAdvisor is optionally deployed via Docker.
03
Prometheus Discovery
Server is registered via file-based service discovery. Prometheus begins collecting metrics within seconds.
04
Dashboard Goes Live
CPU, memory, disk, network, and container metrics populate your real-time dashboard automatically.
05
Alerts & Self-Healing
Alert thresholds go active instantly. The self-healing engine is armed and ready to remediate incidents via SSH.
🔐 Security Guarantee
  • SSH key held in memory only during setup — then garbage collected
  • Keys are never written to disk or stored in the database
  • All remote fix commands are whitelisted — no arbitrary execution
  • Every remediation is logged with attribution and full audit trail
